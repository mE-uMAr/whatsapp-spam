# WhatsApp Auto-Sender (Selenium)

Send scheduled WhatsApp messages via **Selenium + Chrome**.

⚠️ For personal/educational use only. Don’t spam.

---

## Setup

```bash
git clone https://github.com/mE-uMAr/whatsapp-spam.git
cd whatsapp-spam
```

---

## Usage

```bash
uv run spam.py
```

It will prompt:

```bash
Enter phone number (e.g. 923001234567):
Enter message to send:
How many times?:
```

✅ First run: scan QR on WhatsApp Web.
✅ Session saved in `./whatsapp_profile` for next runs.
